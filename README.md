# Clínica Vida Plena

Projeto desenvolvido como desafio do módulo de HTML da DIO.

## Objetivo

Criar um site quase completo para uma clínica médica, utilizando os recursos aprendidos no módulo, como:

- Estruturação de páginas HTML
- Formatação de texto
- Imagens
- Tabelas
- Formulários
- Iframe
- Menu de navegação

## Páginas do projeto

- Página Principal
- Sobre a Clínica
- Horário de Atendimento
- Contato

## Tecnologias utilizadas

- HTML5
- CSS3 básico
